Excel Import Guide

Follow these instructions to set up your Excel files for importing data into OwlJudge.

Users Import

  • Columns Required: Email, Name, Role, Password (optional)
  • Email: Must be unique.
  • Role: Must be one of: user, judge, admin.
  • Password: If not provided, a default password will be set.

Projects Import

  • Columns Required: Project ID, Project Name, Group ID
  • Project ID: Must be a unique integer.
  • Group ID: Must be an integer.

Scores Import

  • Columns Required: Score ID, Category, Score Given, Project ID, Judge ID
  • Score ID: Must be a unique integer.
  • Category: Text description of the score category.
  • Score Given: Integer value between 0 and 100.
  • Project ID: Must match an existing Project ID.
  • Judge ID: Must match an existing Judge's User ID.

Ensure that all required columns are present and that data types are correct to avoid import errors.
